git branch --> show total branch currently
git branch branch_name --> add a new branch with branch_name 
git checkout branch_name --> switch to branch branch_name
git merge branch_name--> merge branch_name to current branch  
git branch -d branch_name --> delete branch "branch_name"
git branch -m master --> rename current branch name to master 
git checkout -b  branch_name --> add a new branch (sibling branch of the previous commit)


git log --all --decorate --oneline --graph --> to see all branchings
